## Image Guidelines
- **Format**: Use JPG, PNG, or WebP formats
- **Size**: Slideshow images should be 1200x600px, product images should be square (at least 300x300px)
- **Quality**: Use high-quality images for best appearance
- **Naming**: Use exact filenames as shown above
- **Fallback**: If images are missing, the site will show emoji icons

## How to Update
1. Place your images in this folder with the exact names shown above
2. Refresh your website to see the changes
3. If an image is missing, the site will automatically show a fallback icon

## Tips
- Use bright, clear images for products
- Ensure good lighting for product photos
- Show products in their packaging when possible
- Use consistent image sizes for professional appearance
